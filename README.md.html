# LinUINO Basierend auf der Arbeit von Thorsten Voss (TonUINO) ## Installation Alle zentralen Funktionen sind über eine `Makefile` abgebildet. Die Ordner für die Quelle und das Ziel der SD-Karte müssen dabei als Environment-Variablen übergeben werden. ~~~~~~ sh CARDSRC=/path/to/src CARDDST=/path/to/dst make copy ~~~~~~ oder per `export`: ~~~~~~ sh export CARDSRC=/path/to/src make convert ~~~~~~ In diesem Zusammenhang bietet sich z.B. das Tool [`direnv`](https://github.com/direnv/direnv) an. Die einzelen Tasks der `Makefile` sind: ~~~~~~ none setup: PlatformIO + Bibliotheken installieren dummysongs: Erstellt einige Dummysongs für Testzwecke convert: Convertiert alle `*.m4a` Dateien unter `$CARDSRC` stats: Zeigt Statistiken über die Codebasis an copy: Kopiert die `*.mp3` Dateien auf die SD-Karte (`$CARDDST`) clean: Räumt Buildprodukte auf audio_messages: Erstellt die Audioansagen deploy: Kompiliert den aktuellen Sketch, lädt ihn hoch und öffnet Monitor monitor: Öffnet den Debug-Monitor upload: Kompiliert den aktuellen Sketch und lädt ihn hoch ~~~~~~ ## Ordnerstruktur Die Ordnerstruktur orientiert sind an einem Standardaufbau für ein C++ Projekt. Alle Sourcen liegen unter `./src`. In `./src/main.cpp` liegt der Hauptsketch, der möglichst sauber gehalten ist um die Struktur des Programflows direkt zu zeigen. ~~~~~~ none . ├── .pio # Buildprodukte ├── bin # Skripte (werden per `make` aufgerufen) │   ├── ... │   └── showstats ├── doc # Icons + PDF ├── msg │   └── audio_messages.txt # Audiomessages ├── src # C++ Sourcen │   ├── ... │   └── volume_settings.h ├── tmp # Temporäre Dateien ├── .envrc # Definiert Environment-Variablen ├── .gitignore ├── LICENSE # Lizenzen ├── Makefile # Makefile ├── README.md.html # Diese Datei └── platformio.ini # PlatformIO KonfigurationK ~~~~~~ ## Resourcen * [Bauanleitung TonUINO](https://www.voss.earth/tonuino/) * [Forum TonUINO](https://discourse.voss.earth/) # Schaltplan *************************************************************************************************** * * | | * .---------+-+----------. * | o JD * * SK *-+-----. .---------------------. * | o NC SPEAKER SK *-+----. | | | * | o NC DL *-+---. | '------+-* -
